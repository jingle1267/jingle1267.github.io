3.1 图示

3.2 形象

3.3 主要特征

  1. 支持异步请求，不用刷整个页面(Ajax 1998 创建-2006 纳入W3C)
  2. 前端不再是模板
  3. HTML 为骨架，CSS 为外貌，Java和 JavaScript 为交互。

3.4 重要元素

  1. 浏览器兼容
  2. Ajax
  3. jQuery

4.3 主要特征

  1. 页面变得越来越复杂
  2. 对用户体验要求越来越高
  3. 前端开发变立体
  4. 模块化/工程化
  5. 技术后端化(MVVM，nginx，lua)
  6. 后端微服务 / 前端更多业务逻辑

  1. Angular/React/Vue
  2. MVVM
  3. 前端更多的原生能力

  5. CommonJS / AMD(requireJS) / CMD(SeaJS) / ES6
